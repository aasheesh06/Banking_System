# Banking System

Console based Banking System built using Java, JDBC and MySQL.

## Features
- User Registration
- Login
- Open Account
- Deposit Money
- Withdraw Money
- Check Balance

## Technologies Used
- Java
- JDBC
- MySQL
- MySQL Connector/J

## How to Run
1. Clone the repository
2. Open the project in IntelliJ or any Java IDE
3. Setup MySQL database
4. Update database username and password in the code
5. Run the BankingApp.java file
